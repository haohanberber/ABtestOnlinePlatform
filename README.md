# AB测试在线平台

这是一个轻量的前端版 AB 测试平台，提供常用实验计算能力，并将 AB 测试理论基础沉淀为文档。

## 功能

- AB测试理论基础文档沉淀（见 `AB测试理论基础.md`）
- 页面功能总说明（见 `页面功能说明.md`，变更前必读）
- 项目开发约定（见 `开发约定.md`）
- 实验设计助手（样本量估算 + 实验周期估算）
- A/B转化率显著性检验（双侧两比例 z 检验）

## 使用方式

直接用浏览器打开 `index.html` 即可使用。

## 说明

- 样本量与检验使用正态近似，适用于大样本场景。
- 关键业务实验建议在平台初判基础上，由数据团队进一步复核。
