# 随机分桶算法

以下为本项目采用的随机分桶算法参考实现（Python 版本）：

```python
def mmh3_to_int(s):
    import mmh3
    # return mmh3.hash(s)
    hash_val = mmh3.hash128(str(s), seed=233)
    return hash_val


def assign_ab_group(df, id_col):
    df = df.copy()

    group_map = {
        0: "A0",
        1: "A1",
        2: "A2",
        3: "B",
    }

    # 0,1,2,3 四组，理论上各 25%
    df["ab_group_id"] = df[id_col].apply(lambda x: mmh3_to_int(x) % 4)
    df["ab_group"] = df["ab_group_id"].map(group_map)
    return df
```

说明：
- 默认 `seed=233`，页面中可手动修改。
- 本项目前端实现使用 MurmurHash3 128 位哈希，与上述思路一致。
- 当分桶数为 4 时，组名映射为 `A0/A1/A2/B`；其他桶数使用 `G0/G1/...` 命名。
